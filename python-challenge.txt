Level 1: Absolute Basics (Input, Output, Variables)

1.1 Guessing Game: Generate a random number and have the user guess it, providing hints.
1.2 Temperature Converter: Write functions for Celsius to Fahrenheit and vice-versa.
1.3 Seconds in a Year: Calculate and print the total seconds in 365 days. 

Level 2: Strings & Lists (Loops, Conditionals)

2.1 Count Vowels: Count 'a', 'e', 'i', 'o', 'u' in a given word or sentence.
2.2 Hide Credit Card: Mask all but the last four digits with asterisks (e.g., "****1234").
2.3 SpongeCase: Convert a string to "sPoNgEcAsE".
2.4 heck if Sorted: Determine if a list of numbers is in ascending order.

Level 3: Logic & Data Structures (Functions, Dictionaries)

3.1 X's & O's: Check if a string has an equal number of 'x's and 'o's.
3.2 Binary to Decimal: Convert a binary number (as string or int) to its decimal equivalent.
3.3 Caesar Cipher: Encrypt or decrypt a message by shifting letters.
3.4 Tribonacci Sequence: Generate terms of the Tribonacci sequence.

Level 4: Intermediate Concepts (Modules, Algorithms)

4.1 Raindrops: Convert numbers to corresponding sound strings (Pling, Plonk, Plong).
4.2 Calculator Function: Create a function that performs basic arithmetic (+, -, *, /).
4.3 Pandas (Data): Explore data analysis with DataFrames (for later stages).
